# Energy Tracker App

Track your energy, stress, and productivity patterns throughout the day.

## Quick Deploy to Vercel (Recommended - 2 minutes)

1. **Create a GitHub account** (if you don't have one): https://github.com/signup
2. **Create a new repository**:
   - Go to https://github.com/new
   - Name it `energy-tracker`
   - Click "Create repository"
3. **Upload this folder**:
   - On your new repo page, click "uploading an existing file"
   - Drag and drop ALL files from this `energy-tracker-app` folder
   - Click "Commit changes"
4. **Deploy on Vercel**:
   - Go to https://vercel.com and sign up with GitHub
   - Click "Add New Project"
   - Import your `energy-tracker` repository
   - Click "Deploy"
5. **Done!** You'll get a URL like `energy-tracker-abc123.vercel.app`

## Alternative: Deploy to Netlify

1. Go to https://netlify.com and sign up
2. Drag and drop the `build` folder (after running `npm run build`)
3. Get your live URL instantly

## Run Locally

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Features

- Quick energy check-ins (1-5 scale)
- Track tasks with categories
- Visualize patterns over time
- View daily, weekly, or all-time data
- Data stored locally in your browser
